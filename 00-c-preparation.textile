h1. Preparation for CFWheels Projects

There are a few extra steps of preparation when working with CFWheels projects:

h3. Installing Servers

h4. Install Tomcat

* Tomcat
** Download and install "32-bit/64-bit Windows Service Installer":http://www.trieuvan.com/apache/tomcat/tomcat-7/v7.0.8/bin/apache-tomcat-7.0.8.exe
** Test at "http://localhost:8080/":http://localhost:8080/

h4. Install Railo

* Railo
** Download and install "http://www.getrailo.org/down.cfm?item=/railo/remote/download/3.2.1.000/custom/all/railo-3.2.1.000.war":http://www.getrailo.org/down.cfm?item=/railo/remote/download/3.2.1.000/custom/all/railo-3.2.1.000.war
** Rename railo-3.2.1.000.war to railo.war
** Copy railo.war to "C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps\"
** Test at "http://localhost:8080/railo/":http://localhost:8080/railo/

h4. Install CFWheels

* CFWheels
** Download and install "http://cfwheels.org/download/latest-version":http://cfwheels.org/download/latest-version
** Extract and copy the file to "C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps\railo"
** Test at "http://localhost:8080/railo/":http://localhost:8080/railo/

h4. Install MySQL

* Windows
** Download and install "http://dev.mysql.com/downloads/mirror.php?id=401202":http://dev.mysql.com/downloads/mirror.php?id=401202
